Tables & filters

A dense list page is the bread and butter of a console — services, deployments, invoices, audits, members. This chapter goes deep on the table primitive (densities, cell types, sorting, expansion, bulk select, empty / loading) and on the filter vocabulary that lives above it (filter chips, query bar, saved views, column filters).

Anatomy

A list page is built from five horizontal bands. Top to bottom: page header, filter strip, view tabs, the table itself, footer / pagination. Each is independent and stays at its own visual weight.

Filter chips

A chip is the visual token for "one filter." It's compact, colored when active, dismissible. Shapes for kinds: add (dashed, opens picker), plain (idle key), active (key + value), count (label with badge), group (key with multiple values).

NxTable — enterprise data table

A real JS component, not markup. Drive it with a data array and columns defs; get sorting (multi-column with Shift), per-column filters, global search, drag-to-reorder headers, drag-to-resize widths, pin left/right, hide/show columns, density toggle, row selection + bulk actions, pagination, inline editing, row expansion, CSV export, and persisted UI state — all out of the box. State persists to localStorage under the persistKey you provide.

Basic — sort, search, paginate

Minimum config: pass data + columns. Click a header to sort; hold Shift to add a second sort key.

Source 
new NxTable(el, {
  data, columns: [
    { key:'id',     label:'ID',     type:'text',   width: 110, pinned:'left' },
    { key:'name',   label:'Name',   type:'text',   width: 200 },
    { key:'role',   label:'Role',   type:'text',   width: 130 },
    { key:'salary', label:'Salary', type:'money',  width: 130, align:'right' },
    { key:'hired',  label:'Hired',  type:'date',   width: 160 },
  ],
  pageSize: 8,
});

Enterprise — selection, bulk, filter, density, columns, export

Drag headers to reorder. Drag the right edge of a header to resize. Use the column-menu (⋯) to pin, hide, or auto-size. Filter (▽) on any column header — text, multi-select, or numeric range. Toolbar exports a CSV of what's currently visible.

Inline edit + row expand + pinned columns

Double-click an editable cell to edit (Enter commits, Esc cancels). Click the chevron to expand a row into a detail panel.

Right-click context menu + drag-to-select

Right-click any row for a context menu — copy, expand, bulk actions, select-all-on-page, invert, clear. Click and drag on the table body to draw a selection box; hold Shift/⌘ while dragging to add to the existing selection. Shift-click a row for range select. ⌘/Ctrl-click to toggle. Custom context menus via contextMenu: (row) => items[].

API reference

Constructor options

dataRow[] — array of row objects.
columnsColumn[] — see below.
getRowId(row, i) => id. Defaults to row.id ?? i.
selectable'multi' | 'single' | false
density'compact' | 'normal' | 'comfy'
paginate / pageSizeBoolean and Number (default 20).
searchToggle the global search input in the toolbar.
showToolbarHide the whole top bar if false.
persistKeyString — saves widths, hidden, pinned, sort, density to localStorage.
bulkActionsString[] — labels for the bulk action sheet.
expandable / expandRenderEnable per-row expand; renderer returns the inner HTML.
rowClickSelectsClick a row to select (default true). Honors Shift/⌘ modifiers.
lassotrue | false | 'auto' — drag-to-select box. Auto enables it for multi-select.
contextMenu(row, table, {selectedRows}) => items[] — customize the right-click menu. Pass false to disable.
onAction(action, payload) => void — fires on bulk, edit, copy, and custom actions.

Column shape

keyField name on row.
labelHeader text.
typetext | number | money | date | status | bar | tags | custom — drives default rendering + sort.
render(value, row) => htmlString. Overrides type rendering.
width / minWidthInitial px / floor for resize.
pinned'left' | 'right' | null
sortableDefault true.
filterable / filterKindToggle the funnel button. 'text' | 'select' | 'range'.
optionsString[] — choices when filterKind === 'select'.
toneMapMap status value → pill tone (ok | warn | err | info | neutral).
editableAllow double-click inline edit.
align'left' | 'right' | 'center'
hiddenHide by default (still in columns menu).

Instance methods

setData(rows)Replace dataset, reset pagination.
addRow(row)Append a row.
removeRows(ids)Delete by id list.
getSelectedRows()Returns currently selected row objects.
clearFilters()Resets global + column filters.
exportCSV()Triggers a CSV download of the filtered/sorted dataset.
refresh()Re-renders from current state.
