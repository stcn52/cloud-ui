Part 01 · Foundations

Design tokens, color, type & space.

NextCli uses a cool blue-tinted neutral ramp with a single azure accent. Everything on this page is expressed as CSS custom properties in tokens.css.

Principles

Three rules that shape every decision below.

01 Calm over loud

Cool hues, hairline borders, tight typography. Color is spent only on state and the single accent — everything else stays neutral so data can speak.

02 Density earns trust

This is a tool for operators. 13px base, 34px rows, 1px lines. Every px of padding must justify itself.

03 Monospace for truth

IDs, counts, SHAs, durations — anything a user might copy, compare, or grep — renders in JetBrains Mono with tabular numerals.

Color

All colors are defined in the OKLCH space so we get perceptually even lightness ramps across hues.
